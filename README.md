# bitinfo

JSON bitcoin info grabber

## Getting Started

### Features

Converts to AUD

### Prerequisites

```
python3
```

### Installing

Clone the Repository and cd into it

```
git clone http://github.com/tealkine/bitinfo.git && cd bitinfo
```

Run bitinfo

```
python3 bitinfo.py
```
