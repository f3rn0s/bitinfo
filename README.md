# bitinfo

JSON bitcoin info grabber

## Getting Started

### Features

Converts to AUD

### Prerequisites

```
python3
requests
```

### Installing

Clone the Repository and cd into it

```
git clone http://github.com/f3rn0s/bitinfo.git && cd bitinfo
```

Run bitinfo

```
python3 bitinfo.py
```
